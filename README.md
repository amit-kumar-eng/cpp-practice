# C++ Practice Repository 🚀

This repository contains my C++ learning journey. I'm practicing C++ concepts from basics to advanced, one program at a time.

## 📁 Programs Included
1. *even_odd.cpp* - Checks if a number is even or odd
2. *grade.cpp* - Calculates student grade from marks: A/B/C/Fail
3. *firstprogram.cpp* - My first C++ program - Hello World
4. *secondprogram.cpp* - Basic input/output practice
5. *sum_of_two_numbers.cpp* - Adds two numbers

## 💻 How to Run
1. Download the code
2. Compile using: g++ filename.cpp -o output
3. Run using: ./output on Mac/Linux or output.exe on Windows
